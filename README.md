## What is Post-Quantum-Cryptography (PQC) and why is it important?
PQC is a set of cryptographic algorithms that are designed to resist attack by quantum computers, which will be much more powerful than classical computers. Quantum computers right now don’t really exist, only in a small and unstable form in strictly scientific environments. 
Current algorithms are based on mathematic problems that a classic computer takes extremely long to solve, but for quantum computer it would be computationally easy and faster. That is why it could break current cryptography.
It is important to consider this now, even though quantum computers are not realized yet and could most likely need another few years to be operational, because attackers can then start using it to decrypt already collected data. Known as “harvest now, decrypt later”. So, they already starting now to collect data.
## Current development of PQC
Currently governments and standardization organisation are initiating a rapid evolvement of PQC algorithms. Some standards are already set and some institutions with critical data (like banks, etc.) starting to implement PQC. But it is still at the beginning, and the solutions are still in need of improvement. One of the biggest player in this standardization process right now is National Institute of Standards and Technology (NIST).

There are different approaches where we can categorize algorithms and one is code-based cryptography. Where the underlying theory revolves around "Linear Codes".

### Code-based cryptography
includes cryptographic systems which rely on error-correcting codes. Error correction code is in computing, telecommunication, information theory, and coding theory, a technique used for controlling errors in data transmission over unreliable or noisy communication channels. (also called forward error correction (FEC) or channel coding). In coding theory, a linear code is an error-correcting code for which any linear combination of codewords is also a codeword.
### Mathematical Coding
translating mathematical formulas, structures, and algorithms into computer code, often using languages like Python, ...
### Coding theory
is the study of encoding information into different Symbols
(Difference to Cryptography: In coding theory, we ignore the question of who has access to the code and how secret it may be. Instead, one of the primary concerns becomes our ability to detect and correct errors in the code)
### What is linear codes?
type of error-correcting code; codewords are formed as linear combinations of message bits (typically vectors over a finite field); linear combination of codewords is also a codeword
